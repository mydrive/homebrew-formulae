# Homebrew Formulae

[Homebrew][1] formulae for our packages.


## Usage

```shell
$ brew tap mydrive/formulae
$ brew install <formula>
```

To update the brew packages, use brew update first:

```shell
$ brew update
$ brew upgrade mydrive
```


## Available Formulae

- gsl@1 ported from the [removed formula][2]
